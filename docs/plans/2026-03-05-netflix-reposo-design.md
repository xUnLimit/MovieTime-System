# Netflix Reposo Module Design

**Date:** 2026-03-05
**Status:** Approved

## Overview

New module to track Netflix accounts in a rest/dormant state (28-31 days). Uses the existing `servicios` collection with additional fields.

## Data Model

New optional fields on `Servicio` type:

```typescript
enReposo?: boolean;          // true when in reposo
diasReposo?: number;         // 28-31 days selected
fechaInicioReposo?: Date;    // when reposo started
fechaFinReposo?: Date;       // fechaInicioReposo + diasReposo
```

Rules:
- `enReposo: true` → `activo: false` (automatic)
- Hidden from Servicios module and venta selection
- Still counts for financial metrics (pagos, gastos)

## Form Changes (Crear/Editar Servicio)

- If category = Netflix → Estado dropdown shows "Reposo" option
- Selecting Reposo → shows "Duración del reposo" field (28-31 days)
- fechaInicioReposo = service start date
- fechaFinReposo = fechaInicioReposo + diasReposo

## Page: /netflix-reposo

Layout: Header + breadcrumb + 3 metric cards + DataTable

### Metric Cards
| Card | Logic | Color |
|------|-------|-------|
| En Proceso | enReposo && fechaFinReposo > today | Blue |
| Próximos a Finalizar | diasRestantes <= 7 && > 0 | Yellow |
| Completados | fechaFinReposo <= today | Green |

### DataTable Columns
Nombre, Email, Fecha Inicio, Fecha Fin, Días Restantes, Progreso (progress bar), Estado, Acciones

### Actions
1. **Activar Netflix** — enReposo: false, activo: true, clear reposo fields
2. **Activar y Renovar Netflix** — same + open PagoDialog

## Notifications

- New tab "Netflix Reposo" in Notificaciones module
- Shows in NotificationBell (header)
- Generated by notificationSyncService when fechaFinReposo <= today
- Stays as "Completado" until manual activation

## Filters

- Servicios module: filter out enReposo === true
- VentasForm: already filters activo === true (auto-excluded)
- Category counters: not counted as serviciosActivos (activo: false)

## Sidebar

Under OTROS section with Moon icon, before Editor de Mensajes.

## Technical Patterns

- useServerPagination with filter enReposo == true
- Metrics via getCount() (free)
- getCurrencySymbol() for currencies
- Activity log for activate/renew actions
